Developpement web

Une application web est un programme qui peut s'exécuter dans un navigateur. Dans cette leçon, les enfants apprennent la base du développement d'application web.

La robotique

Dans ce cours, les enfants sont initiés aux robots. Ils apprennent à connaître le genre de choses que font les robots, l'excitation qui accompagne la conception et les essais, et ce qui inspire les idées pour différents robots.
